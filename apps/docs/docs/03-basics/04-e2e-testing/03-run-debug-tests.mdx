1. Install 'Playwright Test for VSCode' extension

2. Make sure that the servers and the app you want to test is running

```
  nf start
```

```
 nx serve journeys
```

3. Make sure that only firefox is running in playwright config

- go to `playwright.config.ts`
- make sure firefox browser is uncommented
- make sure other browsers are commented out

4. Run all playwright tests (e.g journeys)

```shell
    pnpx nx run journeys-e2e:e2e
```

5. Run just one playwright tests

```shell
    pnpx playwright test apps/journeys-e2e/src/e2e/journeys.spec.ts
```

6. Devcontainer's HTML results can be found at the root level in playwright-report/index.html.
   Copy entire directory to local and open the html report.

More resources on running tests: https://playwright.dev/docs/running-tests
